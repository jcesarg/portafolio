Diseño web

Planificación, idea, creación e implementación de interfaces y plataformas digitales. Proceso que engloba todos los elementos necesarios para crear páginas web.

Diseño gráfico

Combinar texto e imágenes con el fin de transmitir un mensaje visual. Se emplea, sobre todo, en el marketing y la comunicación corporativa.

UI/UX

El diseño UX hace referencia a lo que experimenta el usuario antes, durante y después de entrar en contacto con una página web. UI es el conjunto de elementos de la pantalla que permiten al usuario interactuar con una página web.

SEO
